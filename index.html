Open NCU

NCU API

We provide variety of APIs about NCU, even personal information via NCU OAuth.

NCU OAuth

A web interface for NCU OAuth and NCU API. Users are able to check or revoke permissions and developers could register clients to get tokens to access NCU API.

NCU App Store

A platform exhibit apps or services developed by NCU members.

Repositories

Most of our projects are under open-source, including api services, android apps and web applications.
